# Microservices

**WIP.** Experimental sandbox of a fullstack web project with a microservice / microfrontend architecture.

## Setup

```
$ make install
$ make dev / make start
```

## Main Tools

- Docker
- Docker-Compose
- Make
- NextJS
- Nginx
- NodeJS
- React

## Local Features

- Gateway proxy with Nginx
- Working live reload
